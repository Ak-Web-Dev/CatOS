# MAK.OS
<p>MAK OS is a Web based OS designed to fit in your web! Just for fun. This project doesn't has any real features of a real operating system, but it showcases it is possible for Web-based computing to replace Offline Computing!</p><hr />
<h3><u>Some of the Features are:</u></h3>
<p>-> Fully Functional Window-grabbing System made from JS.</p>
<p>->  Subtle taskbar app icons lines to show active windows status, like windows 11.</p>
<p>->  Clock in Taskbar.</p>
<p>->  2 WebViewer apps to show any content from the web as an app, url is edited from settings.</p>
<p>->  Some Nice Wallpapers you can switch through, via settings.</p>
<p>->  Log In Screen, Logout function and Sleep Function.</p>
<p>->  Username displayed in Start Menu.</p>
<p>->  A Web Browser app in the form of my very own MAK_Tab project.</p>
<p>->  Desktop Icons.</p>
